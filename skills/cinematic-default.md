# Cinematic Default Story Skill

Use this skill when the generated video should feel like a grounded cinematic sequence.

Rules:

- Keep character identity, clothing, age, and visual style stable across scenes.
- Use clear scene geography and visible cause-and-effect between shots.
- Preserve the original plot beat in every scene.
- Use concise camera language: establishing shot, tracking shot, close-up, reveal, or pullback.
- Keep lighting, color, and texture consistent.
- Avoid subtitle cards, web pages, dashboards, and large readable text.

Avoid:

- Comedy beats unless requested.
- Overly abstract symbolism that hides the story.
- Unmotivated changes of location, costume, or character appearance.
